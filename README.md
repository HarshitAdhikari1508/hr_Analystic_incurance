# 📊 Insurance HR Analytics Dashboard

## 📝 Project Overview
This project involves a comprehensive analysis of an insurance company's workforce. Using **Power BI**, I visualized data for 1,000 agents to uncover actionable insights regarding sales performance, employee retention, and the ROI of training programs.

## 🚀 Key Insights
* **Correlation:** Proved that higher **Training Hours** directly impact **Policies Sold**.
* **Performance Lifecycle:** Identified that agents reach peak productivity between **24-48 months** of tenure.
* **Service Quality:** Balanced high sales volume with a steady **3.71/5 CSAT score**.
* **Attrition:** Visualized retention trends to help HR stabilize the workforce.

## 🛠️ Tech Stack & Skills
* **Tool:** Microsoft Power BI Desktop
* **Language:** DAX (Data Analysis Expressions) for custom metrics.
* **Concepts:** Data Modeling, Trend Analysis, UI/UX Design (Neon Dark Theme).
